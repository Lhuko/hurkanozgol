Hello everyone, the Folder named arduinoxml (without .xml extension!!) in our folder is cascade.
It contains the necessary things for the .xml file that I created via.
There is also the arduino.xml file in the Classifier folder.
In the p file, the images are cropped and arranged.
In the detection file, there are images that are tested and detected via cascade.
The only thing you will need to run with the code is the ardunio.xml file.
